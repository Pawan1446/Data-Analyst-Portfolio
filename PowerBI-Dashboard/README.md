# Power BI Sales Dashboard

## Overview

Interactive sales dashboard created in Power BI to analyze business performance.

## Tools Used

- Power BI
- DAX
- Data Modeling
- Data Visualization

## Dashboard Features

- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Sales by Category
- Sales by Product
- Sales by Region
- Monthly Sales Trend
- Interactive Slicers

## Key Insights

- Technology category generated the highest sales.
- November recorded the highest monthly sales.
- West region contributed the highest revenue.
- Interactive filters allow dynamic business analysis.
